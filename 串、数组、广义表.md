### 串的存储结构  
**1.串的顺序存储结构**  
```c
#define MAXLEn 255
typedef struct{
    char ch[MAXLEN+1];  //存储串的一堆数组
    int length;  //串的当前长度
}SString;
``` 
用的更多

**2.串的链式存储结构**  
```c
#define CHUNKSIZE 80
typedef struct Chunk{
    char ch[CHUNKSIZE];
    struct CHUNK *next;
} Chunk;

typedef struct{
    Chunk *head,*tail;  //串的头指针和尾指针
    int curlen; //串的当前长度
}LString;  //字符串的块链结构
```  
优点：操作方便  缺点：存储密度较低
